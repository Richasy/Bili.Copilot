哔哩助理 (Bili Copilot) 安装与更新指南

你好！感谢使用哔哩助理。这是一个由社区开发的第三方哔哩哔哩客户端。
本指南将帮助你安装或更新此应用。

=== 首次安装 ===

1. 解压文件
   你下载的是一个压缩包（.zip）。请务必先将它【全部解压】到一个文件夹中。
   !!! 重要：不要直接在压缩包里双击文件运行，那样会导致安装失败 !!!
   （建议解压到一个你不会轻易删除的地方，比如 D盘的某个文件夹）

2. 运行安装脚本
   打开解压后的文件夹，找到名为 "Install.ps1" 的文件。
   鼠标【右键】点击它，选择【使用 PowerShell 运行】。

3. 按照提示操作
   屏幕上会出现一个蓝色的窗口。
   - 如果提示需要管理员权限，请允许。
   - 脚本会自动安装必要的证书和依赖组件。
   - 等待进度条走完，看到“Installation Complete”或类似的绿色提示即可。

4. 开始使用
   安装完成后，你可以在开始菜单中找到“哔哩助理”。

=== 如何更新 ===

更新步骤与首次安装完全一样！
1. 下载新版本的压缩包。
2. 解压到新的文件夹。
3. 右键 "Install.ps1" -> "使用 PowerShell 运行"。
4. 系统会自动识别并更新旧版本，你的设置和登录状态通常会保留。

=== 常见问题 ===

Q: 双击 Install.ps1 变成了记事本打开怎么办？
A: 请务必【右键】点击文件，选择【使用 PowerShell 运行】。

Q: 提示“在此系统上禁止运行脚本”？
A: 这是 Windows 的安全策略。请尝试以管理员身份打开 PowerShell，输入 `Set-ExecutionPolicy RemoteSigned` 并回车，选择 Y，然后再试一次安装。

Q: 安装失败报错？
A: 请确保你的 Windows 版本在 Windows 10 (19041) 或更高。如果是 Windows 11 就更好了。
